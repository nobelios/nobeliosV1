Les acides aminés

I. Test à la ninhydrine :

I.1. Mode opératoire :

Matériel :

  • Tubes à essais propres
  • Solution aqueuse à 0.25% de nihydrine

I.2. Manipulation :

  • On introduit 0.2g d'a-aminoacide dans un tube à essais que l'on dissout dans un minimum d'eau.
  • On introduit quelques gouttes d'une solution aqueuse de nihydrine à 0.25%, et on chauffe légèrement.

I.3. Conclusion :

Le test sera considéré comme positif si une coloration bleue-violacée apparait.

I.4. Remarques :

Il s'agit d'une réaction très sensible, une coloration bleue-violacée se développe si on est en présence d'un acide a-aminé. On observera également une coloration bleue avec certains ß-aminoacides, peptides ou protéines surtout lors du chauffage.

II. Test du complexe cuivrique :

II.1. Mode opératoire :

Matériel :

  • Tubes à essais propres
  • Solution de sulfate de cuivre II

Manipulation :

  • Dans un tube à essais on ajoute quelques gouttes de solution de sulfate de cuivre II à une solution aqueuse d'acide aminé.

II.2. Conclusion :

Le test sera considéré comme positif si une coloration bleue intense se produit.
